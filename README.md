# vtk-pointcloud-tools

A collection of Python scripts and tools for point cloud processing and visualization using the VTK library.  
This project includes scripts for reading PLY files, applying rigid transformations, setting point colors, batch processing, interactive operations, format conversions, and visualizing results.
